## Cara Install dan Setup

Clone atau Download repository ini. 
Dibutuhkan `node` dan `npm` yang telah terinstall secara global pada komputer atau laptop anda.  

Install:

`npm install`  

Untuk menjalankan website:

`npm run dev`  

Untuk mengunjugi website:

`localhost:3000/`
